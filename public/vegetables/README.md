# Vegetable Images

This folder contains images for vegetables in the CRM system.

## Usage

When adding or editing vegetables as a broker/farmer, you can reference these images using the path:
```
/vegetables/[image-name].jpg
```

For example:
- `/vegetables/tomato.jpg`
- `/vegetables/potato.jpg`
- `/vegetables/carrot.jpg`

## Image Guidelines

- **Format**: JPG, PNG, or WebP
- **Size**: Recommended 800x800px (square)
- **File Size**: Keep under 500KB for best performance
- **Naming**: Use lowercase with hyphens (e.g., `red-tomato.jpg`)

## Available Images

Place your vegetable images in this folder. Examples:
- tomato.jpg
- potato.jpg
- onion.jpg
- carrot.jpg
- cabbage.jpg
- spinach.jpg
- broccoli.jpg
- cucumber.jpg
- bell-pepper.jpg
- eggplant.jpg

## Adding New Images

1. Save your vegetable image to this folder
2. Use the image path `/vegetables/[filename]` in the Image URL field when creating/editing vegetables
3. The image will be displayed in the vegetable listings and cart
